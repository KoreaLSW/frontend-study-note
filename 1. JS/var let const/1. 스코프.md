# 스코프(Scope)의 차이

## 스코프란?

**스코프(Scope)**는 변수나 함수에 접근할 수 있는 유효 범위를 의미합니다. JavaScript에서 변수를 선언할 때, 그 변수가 어디서 접근 가능한지가 스코프에 의해 결정됩니다.

### 스코프의 종류

- **전역 스코프(Global Scope)**: 코드의 어디서든 접근 가능
- **함수 스코프(Function Scope)**: 함수 내부에서만 접근 가능
- **블록 스코프(Block Scope)**: `{}` 블록 내부에서만 접근 가능

### 스코프가 중요한 이유

1. 변수 이름 충돌 방지
2. 메모리 관리 (스코프가 끝나면 변수 해제)
3. 코드의 안전성과 예측 가능성 향상

## var - 함수 스코프

```javascript
function example() {
  if (true) {
    var x = 10;
  }
  console.log(x); // 10 - 블록 밖에서도 접근 가능
}
```

`var`는 함수 스코프를 가집니다. 함수 내부에서 선언하면 함수 전체에서 접근 가능하고, 블록(`{}`)은 무시됩니다.

## let, const - 블록 스코프

```javascript
function example() {
  if (true) {
    let y = 20;
    const z = 30;
  }
  console.log(y); // ReferenceError: y is not defined
  console.log(z); // ReferenceError: z is not defined
}
```

`let`과 `const`는 블록 스코프를 가집니다. `{}` 블록 내부에서만 접근 가능합니다.
