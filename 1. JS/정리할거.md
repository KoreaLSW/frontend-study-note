# JavaScript 면접 질문 항목 리스트

## 1. 기본 문법과 개념

- 변수 선언: `var`, `let`, `const`의 차이점(**1번에 작성**)
- 데이터 타입: 원시 타입 vs 참조 타입
- 타입 변환: 암시적/명시적 변환
- 연산자: 비교 연산자, 논리 연산자, 삼항 연산자
- 제어문: `if/else`, `switch`, `for`, `while`, `forEach`, `map`, `filter`

## 2. 함수와 스코프

- 함수 선언 vs 함수 표현식
- 화살표 함수(Arrow Function)
- 클로저(Closure) 개념과 활용
- 스코프(Scope): 전역, 함수, 블록 스코프(**1번에 작성**)
- 호이스팅(Hoisting) 동작 원리(**1번에 작성**)
- `this` 키워드의 바인딩과 동작 방식
- `call`, `apply`, `bind`의 차이점

## 3. 객체와 배열

- 객체 생성 방법
- 프로퍼티 접근 및 조작
- 배열 메서드: `map`, `filter`, `reduce`, `find`, `some`, `every`
- 얕은 복사 vs 깊은 복사
- 구조 분해 할당(Destructuring)
- 스프레드 연산자(Spread Operator)

## 4. 비동기 프로그래밍

- 콜백 함수와 콜백 지옥
- Promise: 생성, 체이닝, `Promise.all`, `Promise.race`
- `async/await` 사용법
- 이벤트 루프(Event Loop) 동작 원리
- 마이크로태스크 vs 매크로태스크
- `setTimeout`, `setInterval` 동작 방식

## 5. ES6+ 최신 문법

- 템플릿 리터럴(Template Literal)
- 기본 매개변수(Default Parameters)
- 나머지 매개변수(Rest Parameters)
- 모듈 시스템: `import`, `export`
- `Symbol`, `Map`, `Set` 자료구조
- 제너레이터(Generator)와 이터레이터(Iterator)

## 6. 객체 지향 프로그래밍

- 프로토타입(Prototype) 체인
- 프로토타입 기반 상속
- ES6 클래스 vs 생성자 함수
- `super`, `extends` 키워드
- `static` 메서드
- 캡슐화, 상속, 다형성

## 7. DOM 조작과 이벤트

- DOM 구조 이해
- DOM 선택 및 조작 메서드
- 이벤트 리스너 등록/제거
- 이벤트 버블링과 캡처링
- 이벤트 위임(Event Delegation)
- `preventDefault()`, `stopPropagation()`

## 8. 에러 처리와 디버깅

- `try...catch...finally`
- 에러 객체와 커스텀 에러
- `throw` 문
- 디버깅 기법과 도구
- 스택 트레이스 이해

## 9. 메모리 관리와 성능

- 메모리 누수 원인과 방지
- 가비지 컬렉션(Garbage Collection)
- 성능 최적화 기법
- 디바운싱(Debouncing)과 쓰로틀링(Throttling)
- 가상 DOM 개념

## 10. 보안

- XSS(Cross-Site Scripting) 방지
- CSRF(Cross-Site Request Forgery) 방지
- `eval()` 사용의 위험성

- 함수형 프로그래밍 개념

- 스택(Stack)과 힙(Heap)
