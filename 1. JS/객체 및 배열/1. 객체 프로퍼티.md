# 객체 프로퍼티 접근 및 조작

JavaScript에서 객체의 프로퍼티를 다루는 다양한 방법을 정리했습니다.

## 1. 프로퍼티 접근 방법

객체의 프로퍼티에 접근하는 방법은 크게 두 가지가 있습니다.

### 1. 점 표기법(Dot Notation)

```javascript
const person = {
  name: "철수",
  age: 30,
};

console.log(person.name); // "철수"
console.log(person.age); // 30
```

### 2. 대괄호 표기법(Bracket Notation)

```javascript
const person = {
  name: "철수",
  age: 30,
  "favorite-color": "blue", // 하이픈이 포함된 키는 대괄호 표기법만 사용 가능
};

console.log(person["name"]); // "철수"
console.log(person["age"]); // 30
console.log(person["favorite-color"]); // "blue"

// 변수를 사용한 동적 접근
const key = "name";
console.log(person[key]); // "철수"
```

## 2. 프로퍼티 추가 및 수정

```javascript
const person = {
  name: "철수",
};

// 프로퍼티 추가
person.age = 30;
person["city"] = "서울";

// 프로퍼티 수정
person.name = "영희";
person["age"] = 25;

console.log(person); // { name: "영희", age: 25, city: "서울" }
```

## 3. 프로퍼티 삭제

```javascript
const person = {
  name: "철수",
  age: 30,
  city: "서울",
};

delete person.age;
delete person["city"];

console.log(person); // { name: "철수" }
```

## 4. 프로퍼티 존재 확인

```javascript
const person = {
  name: "철수",
  age: 30,
};

// in 연산자
console.log("name" in person); // true
console.log("city" in person); // false

// hasOwnProperty 메서드
console.log(person.hasOwnProperty("name")); // true
console.log(person.hasOwnProperty("city")); // false

// undefined 체크 (주의: 프로퍼티가 undefined일 수도 있음)
console.log(person.name !== undefined); // true
```

## 5. 객체 메서드

```javascript
const person = {
  name: "철수",
  age: 30,
  greet: function () {
    return `안녕하세요, ${this.name}입니다.`;
  },
  // ES6 축약 문법
  introduce() {
    return `제 이름은 ${this.name}이고, 나이는 ${this.age}입니다.`;
  },
};

console.log(person.greet()); // "안녕하세요, 철수입니다."
console.log(person.introduce()); // "제 이름은 철수이고, 나이는 30입니다."
```

## 6. 객체 순회

```javascript
const person = {
  name: "철수",
  age: 30,
  city: "서울",
};

// Object.keys() - 키 배열 반환
console.log(Object.keys(person)); // ["name", "age", "city"]

// Object.values() - 값 배열 반환
console.log(Object.values(person)); // ["철수", 30, "서울"]

// Object.entries() - [키, 값] 배열 반환
console.log(Object.entries(person)); // [["name", "철수"], ["age", 30], ["city", "서울"]]

// for...in 루프
for (const key in person) {
  console.log(`${key}: ${person[key]}`);
}

// Object.entries()와 for...of 루프
for (const [key, value] of Object.entries(person)) {
  console.log(`${key}: ${value}`);
}
```
