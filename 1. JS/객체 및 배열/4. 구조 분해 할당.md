# 구조 분해 할당(Destructuring)

구조 분해 할당은 배열이나 객체의 값을 변수로 추출하는 문법입니다.

## 1. 배열 구조 분해

```javascript
// 기본 사용법
const numbers = [1, 2, 3];
const [a, b, c] = numbers;
console.log(a, b, c); // 1 2 3

// 일부만 추출
const [first, second] = numbers;
console.log(first, second); // 1 2

// 기본값 설정
const [x, y, z = 10] = [1, 2];
console.log(x, y, z); // 1 2 10

// 나머지 요소 수집
const [first2, ...rest] = [1, 2, 3, 4, 5];
console.log(first2); // 1
console.log(rest); // [2, 3, 4, 5]

// 요소 건너뛰기
const [, , third] = [1, 2, 3];
console.log(third); // 3

// 변수 교환
let x1 = 1;
let y1 = 2;
[x1, y1] = [y1, x1];
console.log(x1, y1); // 2 1
```

## 2. 객체 구조 분해

```javascript
// 기본 사용법
const person = {
  name: "철수",
  age: 30,
  city: "서울",
};

const { name, age, city } = person;
console.log(name, age, city); // "철수" 30 "서울"

// 다른 변수명으로 할당
const { name: personName, age: personAge } = person;
console.log(personName, personAge); // "철수" 30

// 기본값 설정
const { name, age, city = "부산" } = { name: "영희", age: 25 };
console.log(city); // "부산"

// 나머지 프로퍼티 수집
const { name, ...rest } = person;
console.log(rest); // { age: 30, city: "서울" }

// 중첩 객체 구조 분해
const user = {
  name: "철수",
  address: {
    city: "서울",
    country: "한국",
  },
};

const {
  name,
  address: { city, country },
} = user;
console.log(name, city, country); // "철수" "서울" "한국"
```

## 3. 함수 매개변수에서 사용

```javascript
// 배열 매개변수
function sum([a, b, c]) {
  return a + b + c;
}
console.log(sum([1, 2, 3])); // 6

// 객체 매개변수
function greet({ name, age }) {
  return `안녕하세요, ${name}님. 나이는 ${age}세입니다.`;
}
console.log(greet({ name: "철수", age: 30 })); // "안녕하세요, 철수님. 나이는 30세입니다."

// 기본값과 함께
function introduce({ name = "익명", age = 0 } = {}) {
  return `${name}님은 ${age}세입니다.`;
}
console.log(introduce()); // "익명님은 0세입니다."
```

## 4. 실전 활용 예시

```javascript
// 배열에서 최대값과 최소값 추출
const numbers = [10, 5, 20, 15];
const [max, ...rest] = numbers.sort((a, b) => b - a);
console.log(max); // 20

// API 응답 처리
const response = {
  status: 200,
  data: {
    users: [
      { id: 1, name: "철수" },
      { id: 2, name: "영희" },
    ],
  },
};

const {
  data: { users },
} = response;
console.log(users); // [{ id: 1, name: "철수" }, { id: 2, name: "영희" }]
```
