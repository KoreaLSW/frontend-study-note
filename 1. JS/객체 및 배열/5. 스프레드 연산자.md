# 스프레드 연산자(Spread Operator)

스프레드 연산자(`...`)는 배열이나 객체를 펼쳐서 사용할 수 있게 해줍니다.

## 1. 배열에서 사용

```javascript
// 배열 복사
const arr1 = [1, 2, 3];
const arr2 = [...arr1];
console.log(arr2); // [1, 2, 3]

// 배열 합치기
const arr3 = [1, 2, 3];
const arr4 = [4, 5, 6];
const combined = [...arr3, ...arr4];
console.log(combined); // [1, 2, 3, 4, 5, 6]

// 배열에 요소 추가
const newArr = [0, ...arr3, 4];
console.log(newArr); // [0, 1, 2, 3, 4]

// 함수 인수로 전달
const numbers = [1, 2, 3, 4, 5];
console.log(Math.max(...numbers)); // 5
console.log(Math.max(1, 2, 3, 4, 5)); // 위와 동일

// 배열 복사 및 수정
const original = [1, 2, 3];
const modified = [...original, 4, 5];
console.log(original); // [1, 2, 3] - 원본은 변경되지 않음
console.log(modified); // [1, 2, 3, 4, 5]
```

## 2. 객체에서 사용

```javascript
// 객체 복사
const obj1 = { name: "철수", age: 30 };
const obj2 = { ...obj1 };
console.log(obj2); // { name: "철수", age: 30 }

// 객체 합치기
const obj3 = { name: "철수", age: 30 };
const obj4 = { city: "서울", country: "한국" };
const merged = { ...obj3, ...obj4 };
console.log(merged); // { name: "철수", age: 30, city: "서울", country: "한국" }

// 프로퍼티 덮어쓰기
const base = { name: "철수", age: 30 };
const updated = { ...base, age: 31 };
console.log(updated); // { name: "철수", age: 31 }

// 프로퍼티 추가
const newObj = { ...base, city: "서울" };
console.log(newObj); // { name: "철수", age: 30, city: "서울" }
```

## 3. 나머지 매개변수(Rest Parameters)

함수에서 사용할 때는 나머지 매개변수로도 사용됩니다.

```javascript
// 나머지 매개변수
function sum(...numbers) {
  return numbers.reduce((acc, n) => acc + n, 0);
}
console.log(sum(1, 2, 3, 4, 5)); // 15

// 일반 매개변수와 함께
function greet(greeting, ...names) {
  return `${greeting}, ${names.join(", ")}!`;
}
console.log(greet("안녕하세요", "철수", "영희", "민수")); // "안녕하세요, 철수, 영희, 민수!"
```

## 4. 구조 분해와 함께 사용

```javascript
// 배열에서 나머지 요소 수집
const [first, second, ...rest] = [1, 2, 3, 4, 5];
console.log(first); // 1
console.log(second); // 2
console.log(rest); // [3, 4, 5]

// 객체에서 나머지 프로퍼티 수집
const { name, ...restProps } = { name: "철수", age: 30, city: "서울" };
console.log(name); // "철수"
console.log(restProps); // { age: 30, city: "서울" }
```

## 5. 실전 활용 예시

```javascript
// React에서 state 업데이트 (불변성 유지)
const [state, setState] = useState({ count: 0, name: "철수" });
setState({ ...state, count: state.count + 1 });

// 배열 중복 제거
const arr = [1, 2, 2, 3, 3, 4];
const unique = [...new Set(arr)];
console.log(unique); // [1, 2, 3, 4]

// 객체 기본값 설정
function createUser(userData) {
  return {
    name: "익명",
    age: 0,
    city: "서울",
    ...userData,
  };
}

const user = createUser({ name: "철수", age: 30 });
console.log(user); // { name: "철수", age: 30, city: "서울" }

// 배열 복사 및 수정 (불변성)
const todos = [
  { id: 1, text: "공부하기", done: false },
  { id: 2, text: "운동하기", done: false },
];

// 특정 항목 업데이트
const updatedTodos = todos.map((todo) =>
  todo.id === 1 ? { ...todo, done: true } : todo
);
```

## 6. 스프레드 연산자 주의사항

```javascript
// 얕은 복사만 수행
const original = {
  name: "철수",
  address: {
    city: "서울",
  },
};

const copied = { ...original };
copied.address.city = "부산";
console.log(original.address.city); // "부산" - 원본도 변경됨!

// 깊은 복사가 필요하면
const deepCopied = {
  ...original,
  address: { ...original.address },
};
deepCopied.address.city = "부산";
console.log(original.address.city); // "서울" - 원본은 변경되지 않음
```
